# Ping Test

## What is it?
A simple script that measures network latency (ping) and displays the result in milliseconds (ms).

## Why does it exist?
This project was created for learning purposes, focusing on:
- Function return values
- External libraries
- Basic API usage
- Unit conversion (seconds → milliseconds)

Planned improvements
Graphical User Interface (GUI)
Export as executable (.exe)
Destination selection (custom ping target)

## How to use

```bash



### 1. Install dependencies
Make sure you have Python 3.x installed, then run:
pip install ping3

### 2. Run the script
From the project directory, execute:
python main.py

### 3. Output
The script will measure the ping to a public DNS server and display the result, for example:
"your ping is 37.4ms"
